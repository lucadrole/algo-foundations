# Study resources

## Algorithms Illuminated, Part 1

Contents:

- Asymptotics and big-O
- Divide-and-conquer
- Randomized algorithms
- Sorting and selection

### Why

- Learn fast subroutines
- Learn general algo design paradigms
- Algorithmic thinking

A great point in favour of this book is its accessibility. It is not a traditional textbook.

### Suggested further resources

- Mathematics for Computer Science, by Eric Lehman and Tom
Leighton
- Coursera and Stanford Lagunita
- YouTube video playlists from the book website
- **Important**: programming projects to apply concepts.
